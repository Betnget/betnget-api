# API Betnget

API do projeto Betnget

## Pré-requisitos

- Nodejs: v18.17.1
- NPM: 9.6.7

## Instalações de dependências

Para instalar as dependências, execute o seguinte comando:

```bash
npm install
```

## Run

Para iniciar a API execute o seguinte comando:

```bash
npm run dev

ou

npx nodemon
```
